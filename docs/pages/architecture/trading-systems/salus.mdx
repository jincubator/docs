# Salus Architecture

> Historical presentation retained for compatibility. Current Salus Work is at [Salus](/work/salus), and the current reusable architecture is [Designing Evidence-Aware Trading-System Pipelines](/architecture/trading-systems/solver).

Salus evolves the Solver architecture into clearer trading and execution infrastructure.

The architecture emphasizes replay validation, read models, route-profit gates, strategy and execution staging, migration discipline, and productization of trading-system research.
