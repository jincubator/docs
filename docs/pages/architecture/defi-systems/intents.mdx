# Intent Architecture

Intent architecture covers the systems that express desired outcomes, lock or allocate resources, route execution, and validate settlement conditions.

The architecture work connects resource locking, arbiters, fillers, solvers, Uniswap v4 hooks, Tycho execution, and protocol-specific workflows.

## Architecture Lens

- Intent expression, mandate validation, and resource locking.
- Solver and filler participation.
- Rebalancing, keeper execution, hooks, and protocol automation.
- Cross-linking between DeFi protocols and trading-system execution.
