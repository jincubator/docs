# Payments

> Historical topic page retained for compatibility. Current settlement research is [Durable Financial Settlement](/research/financial-infrastructure/settlement/intro).

Payments research focuses on the systems that move value between users, applications, enterprises, banks, wallets, and onchain infrastructure.

The current material is grounded in architecture analysis rather than a standalone payments product. Relevant themes include API platform design, wallet and contract interfaces, fiat and onchain rails, KYC and operational controls, settlement readiness, and production integration patterns.

## Current Focus

- Payment API platforms
- Wallet and contract interaction surfaces
- Fiat and onchain rail integration
- Operational controls and compliance boundaries
- Enterprise platform readiness
