# Protocol Integration

- **Maturity:** Reference architecture
- **Evidence:** [Dated Salus status](/work/salus)

Protocol integration is evaluated at an architectural level:

- state and component-model compatibility;
- deterministic quote or simulation behavior;
- route and asset representation;
- validation and failure boundaries;
- execution-family support and test evidence.

The public summary does not identify current provider choice, fee-selection policy, thresholds, priority rules, or submission configuration.
