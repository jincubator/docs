# Work

John's selected current work is presented as concise project stories with clear contribution boundaries, dated results, limitations, and links into the underlying architecture.

## Salus

[Salus](/work/salus) is an active trading-system research and reference implementation. It explores market-data ingestion, graph construction, candidate-route generation, route evaluation, simulation, and the boundary between a signal and an executable outcome.

The retained results show what has been built and measured without claiming commercial validation, production-proven profitability, or repeatable realized profit.

[Read the Salus project overview](/work/salus).
